# Tienda Online - Carrito de Compras PAPI-SHOP

## Descripción
Esta es una tienda en línea que permite a los usuarios explorar productos y gestionar un carrito de compras. El proyecto incluye el nombre de los productos, interactividad y diseño responsivo.

## Funcionalidades
- **Exploración de productos** tenemos este apartado en cuatro categorías:
  - **Componentes PC**
  - **Accesorios**
  - **Ropa**
  - **Electrodomésticos**
- **Carrito de compras**:
  - Agregar y quitar productos.
  - Visualizar resumen del carrito y el total a pagar.
  - filtro de búsqueda.
- **Páginas adicionales**:
  - Detalle de productos.
  - Formulario de contacto.
  - Login o registro de usuarios.
- **Diseño responsivo**: Optimizado para dispositivos móviles.

## Tecnologías Utilizadas
- **HTML5**: Estructura de la aplicación.
- **CSS3**: Estilos y diseño responsivo.
- **JavaScript**: Lógica del carrito de compras e interactividad.
- **Librerías externas**: Contenidas en la carpeta `lib`tales como Bootstrap.

## Instalación
1. Clona este repositorio:
   ```bash
   git clone https://github.com/Elian46k/proyecto_03_carrito_de_compra.git
   ```
2. Ve al directorio del proyecto:
   ```bash
   cd proyecto_03_carrito_de_compra
   ```
3. Abre `index.html` en tu navegador para iniciar la aplicación.

## Estructura del Proyecto
```
proyecto_03_carrito_de_compra/
├── index.html        # Página principal
├── tienda.html       # Página de la tienda
├── detalle.html      # Página de detalles de producto
├── contacto.html     # Formulario de contacto
├── log.html          # Página de login/registro
├── css/              # Archivos de estilos
├── js/               # Scripts JavaScript
├── img/              # Imágenes de productos y recursos
├── lib/              # Bibliotecas externas
├── .git/             # Control de versiones
└── .vscode/          # Configuración del editor
```

## Uso
1. Abre el archivo `index.html` en cualquier navegador web.
2. Explora las diferentes categorías de productos.
3. Agrega productos al carrito desde la página de la tienda o detalles.
4. Revisa el carrito para confirmar tu selección.
